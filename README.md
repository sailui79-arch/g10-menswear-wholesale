# G-10 男装批发网站

这是一个适合 Vercel 发布的静态网站，重点是让缅甸客户用手机快速打开。

## 每天更新产品图片

1. 把当天产品图片放到 `products/日期/` 文件夹，例如 `products/2026-06-27/`。
2. 打开 `products.js`，新增产品信息。
3. 图片多的时候，首页只放精选款，全部图片可以按日期页面或相册页展示。

## 为什么这样做比较快

- 首页只加载少量精选图片。
- 图片使用懒加载，客户滑到哪里才加载哪里。
- 不使用复杂后台，Vercel 可以把页面缓存到全球节点。
- 后面产品很多时，可以把大图压缩成小图，详情再打开原图。
