# G-10 每日图片更新方法

目标：每天约 500 张产品照片，但让缅甸客户打开首页仍然快。

## 推荐做法

1. 首页只放 8 到 20 张精选新款。
2. 每天新建一个日期文件夹，例如 `products/2026-06-28/`。
3. 大量图片放到日期文件夹，不要一次全部塞进首页。
4. 上传前把手机原图压缩成网页图，推荐宽度 900px 到 1200px。
5. 文件名用英文和数字，例如 `g10-0628-001.jpg`，不要用中文、空格或特殊符号。

## 首页加新款

打开 `products.js`，按这个格式增加：

```js
{
  id: "G10-0628-001",
  name: "男士休闲短袖",
  category: "T 恤",
  note: "多色可选，支持混批",
  image: "./products/2026-06-28/g10-0628-001.jpg"
}
```

## 速度建议

- 真实上线时，优先使用自己压缩后的图片，不要长期依赖外部图片地址。
- 如果每天都要展示 500 张，建议后面升级成“按日期分页相册”：每页 40 到 60 张。
- Vercel 免费版适合先开始，但图片越来越多后，可以把图片放到专门的图片空间，再让网站读取。
